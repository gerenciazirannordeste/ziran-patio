Regras do pátio
Empilhamento padrão: 5 de altura. O tamanho de cada gaveta é proporcional à capacidade (capacidade ÷ 5 = posições no chão).

Renumerar reordena G01, G02... coluna por coluna (esquerda → direita, cima → baixo), como no pátio físico.

O layout é salvo automaticamente neste navegador. Use Exportar JSON para backup.
